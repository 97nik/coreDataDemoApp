# coreDataDemoApp
## Описание
ToDoList
## Решеные задачи
  - Сохранение данных после выхода / закрывания приложения 
  - Возможность редактировать задачи 
  - Удаление задач в листе 
 
## Использованые технологии:
  - UiKit
  - Архитерктура MVC
  - Без использования Storyboard
